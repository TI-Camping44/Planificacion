# Presentación IT - Camping 44 S.A.

Este repositorio incluye una presentación estratégica mensual del área IT en formato **Reveal.js**.

## Archivos

- `index.html` → archivo principal para publicación (GitHub Pages usa este archivo por defecto).
- `deck-it-camping44.html` → acceso alternativo / compatibilidad.

## Ver la presentación en enlace público (GitHub Pages)

1. Ir a **Settings** del repositorio.
2. Entrar en **Pages**.
3. En **Build and deployment**, seleccionar:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` y carpeta `/ (root)`
4. Guardar.
5. Esperar 1–3 minutos.

La URL pública quedará con este formato:

`https://<usuario>.github.io/Planificacion/`

## Ejecución local

Abrir `index.html` directamente en navegador o servirlo con un servidor local.
